Frequently Asked Questions

Text Color Fader is a free online tool that allows you to create gradient text effects. You can input any text, choose your starting and ending colors, and our tool will generate HTML or BBCode that creates a smooth color transition across your text, with each character having its own color in the gradient.

After generating your gradient text, simply click the "Copy to Clipboard" button. Then:

  • For HTML: Paste the code into your website's HTML editor or page builder.
  • For BBCode: Paste the code into forum posts, comments, or any platform that supports BBCode formatting.

The code will automatically display your text with the gradient effect applied.

The HTML output will work on most websites that allow custom HTML, including:

  • WordPress (in HTML/Text mode)
  • Wix (using HTML embeds)
  • Squarespace (in code blocks)
  • Custom websites

The BBCode output is designed for forums and platforms that support BBCode formatting but not HTML.

Some platforms may strip HTML formatting for security reasons, in which case the gradient effect won't appear.

There's no strict character limit for the text you can process. However, very long text (thousands of characters) may:

  • Take longer to process
  • Generate large amounts of code
  • Potentially slow down the page where you use it

For optimal performance, we recommend using gradient effects for headlines, short paragraphs, or other highlight text rather than entire pages of content.

Currently, Text Color Fader doesn't have a built-in save feature. However, you can:

  • Copy and save the generated code in a text document
  • Take a screenshot of your settings for future reference
  • Bookmark the page with your text and color choices in the URL (coming soon)

We're working on adding user accounts with saved designs in a future update.
